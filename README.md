# 本研究について
## テーマ
ユーザーの興味喚起を目的としたインタラクティブキャラクターシステムの研究

## 概要
本研究では、AIによる画像認識技術であるMediaPipe Face Meshを用いて、人の顔の向きや動きをリアルタイムで検出し、その動きに応じてキャラクターが反応するシステムの開発を行っています。<br>
Webカメラで取得した映像から顔の特徴点を解析し、顔を上下左右に動かしたり傾けたりした動作を判定して、キャラクターの表情や動作へ反映させます。人の動きを直感的にキャラクターへ伝えられる仕組みを実現することで、より自然で親しみやすいインタラクションの実現を目指しています。

## 現在の進捗
MediaPipeでの人検出フェーズの実装

## 現在の実装
- FaceDetection
- Face Mash
- HandTracking

## 今後の予定
